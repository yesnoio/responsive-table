Step 3 — Success

Narrow this window below 640 px to see each table reflow into its mobile layout.

A single media query does all the work: it hides the header row, stacks cells, and reveals the per-cell labels — keeping context right next to every value.

@media screen and (max-width: 640px) {
  /* Hide the real header row */
  table.responsive thead { display: none; }

  /* Turn each row into a card */
  table.responsive tbody tr {
    display: block;
    border: 1px solid #ddd;
    border-radius: 6px;
    overflow: hidden;
    margin-bottom: 1rem;
  }

  /* Stack cells vertically; zero out td padding so label sits flush */
  table.responsive tbody td { display: block; padding: 0; }

  /* Remove bottom border on the last cell — the card border covers it */
  table.responsive tbody tr td:last-child { border-bottom: none; }

  /* Stack the spans inside each cell */
  table.responsive td span { display: block; }

  /* Style the label like a column header; owns its own padding now */
  table.responsive td .label {
    display: block;
    padding: 0.25rem 0.875rem;
    background-color: #ddd;
    font-weight: bold;
    text-align: center;
  }

  table.responsive td .data { padding: 0.6rem 0.875rem; }
}
